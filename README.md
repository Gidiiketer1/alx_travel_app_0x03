## Overview
This project enhances the ALX Travel App by adding asynchronous background job processing using Celery and RabbitMQ. Booking confirmation emails are sent without blocking the main application flow.

## Features
- Celery integration with RabbitMQ
- Background email notifications
- Django email backend
- Improved performance and scalability

## Technologies
- Django
- Celery
- RabbitMQ
- SMTP (Django Email Backend)

## How to Run
1. Start RabbitMQ
2. Run Django server
3. Start Celery worker
4. Create a booking to trigger email notification
